# list(input())

- ```split()``` → 공백 단위로 나누기 (문자열 전체를 하나로 인식)
```
mirr = [ list(input().split()) for _ in range(r) ]
>>[['####'], ['#JF#'], ['#..#'], ['#..#']]
```
이건 공백 기준으로 split() 하고 있어서,
각 줄을 ['####']처럼 통째로 리스트에 넣고 있음.

- ```list()``` → 한 글자씩 쪼개기 (문자 단위로 인식)
```
mirr = [list(input()) for _ in range(r)]
>>> [['#', '#', '#', '#'],
 ['#', 'J', 'F', '#'],
 ['#', '.', '.', '#'],
 ['#', '.', '.', '#']]
```
