# dx, dy: 그래프 알고리즘 
상하좌우로 이동하는 원리와 구현 방법

⭐ 격자 탐색 문제에서 매우 자주 나온다! 

```python
dx = [-1, 0, 1, 0]  # x좌표 변화
dy = [0, 1, 0, -1]  # y좌표 변화
```
| i | 방향 | dx\[i] | dy\[i] | 의미      |
| - | -- | ------ | ------ | ------- |
| 0 | 상  | -1     | 0      | 위로 한 칸  |
| 1 | 우  | 0      | 1      | 오른쪽 한 칸 |
| 2 | 하  | 1      | 0      | 아래로 한 칸 |
| 3 | 좌  | 0      | -1     | 왼쪽 한 칸  |


⬇️

⬇️

이걸 반복문으로 돌린다면....

```python
x, y = 2,2

for i in range(4):
  nx = x + dx[i]
  ny = y + dx[i]

  print(f'{i}번 방향 -> 이동한 위치: ({nx}, {ny})')
```
```
0번 방향 → 이동한 위치: (1, 2)  # 상
1번 방향 → 이동한 위치: (2, 3)  # 우
2번 방향 → 이동한 위치: (3, 2)  # 하
3번 방향 → 이동한 위치: (2, 1)  # 좌
```
